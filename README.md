# playlab-whatsapp-bridge
This project develops a specialized educational bridge connecting Playlab.ai’s steerable AI ecosystem with the Turn.io platform to democratize access to high-quality learning in low-bandwidth regions. It will enable educators to deploy custom AI agents directly to student via mobile devices (i.e. WhatsApp).

## Secret safety
- Do not commit `.env` or any real keys.
- Install pre-commit hooks to scan for secrets:
  - `pip install pre-commit`
  - `pre-commit install`
  - `pre-commit run --all-files`
